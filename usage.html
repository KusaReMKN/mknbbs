使い方など

MKNBBS とはなに？

MKNBBS は KusaReMKN によるインターネット掲示板です。 だれでも自由に匿名で書き込むことができます。 発信元は一切わかりません。

使い方は？

スレッドに書き込む

  • スレ一覧から興味のあるスレッドに入ります。

  • ページの下部に投稿フォームがあります。

  • ハンドルネームは投稿者の名前として書き込みに表示されるものです。 省略すると “名無しの暇人🍊” と表示されます。

  • メッセージ欄に投稿内容を入力します。 Markdown を利用して投稿内容をスタイリングできます。

  • “書き込む” ボタンを押してスレッドに書き込みます。

スレッドを立てる

  • スレ一覧に移動します。

  • ページの下部にスレ立てフォームがあります。

  • 基本的にはスレッドに書き込む際と同様です。 スレッド名もまた入力してください。

  • “スレッドを立てる” ボタンを押してスレッドを立てます。

Markdown 表記について

MKNBBS ではほとんどの Markdown 記法が使えます。 詳細はこのページを参考にしてください。 HTML タグは反映されません。

例えば、以下のような Markdown 表記に対応しています。

斜体・太字・取り消し線

入力

*斜体 italic*
**太字 bold**
***太字斜体 bold-italic***
~~取り消し線 strikethrough~~

表示

斜体 italic 太字 bold 太字斜体 bold-italic 取り消し線 strikethrough

リンク・画像

入力

[トップ](https://www.kusaremkn.com/)

![バナー画像](https://www.kusaremkn.com/img/banner.png)

表示

トップ
